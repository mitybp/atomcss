Atom.css Carbon Gem Icons

Background, text and border colors

Install

Add in <head> tag the Atom Carbon import: <link href="https://atomcss.vercel.app/carbon/docs/1.0/carbon.css" rel="stylesheet">

<!DOCTYPE html>
<html lang="en">
  <head>
      <meta charset="utf-8">
      <meta name="viewport" content="width=device-width, initial-scale=1">
      <title>Atom Carbon demo</title>
      <link href="https://atomcss.vercel.app/carbon/1.0/carbon.css" rel="stylesheet">
  </head>
  <body>
      <h1>Your code here!</h1>
  </body>
</html>

Usage

Write in a HTML element class ac-['text' or 'border']-[color] to add a text color or a border color, if you want a background, write ac-[color] in the element class.
